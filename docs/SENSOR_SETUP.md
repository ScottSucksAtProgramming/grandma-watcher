# Sensor Node Setup Guide

> **Phase 2 — Not yet implemented.**

Phase 2 adds load cell and vitals sensor nodes via Pi Zero W units. This guide will cover setup and wiring when that phase begins.

For current architecture, see `PRD.md` §8 (Sensor Nodes) and §13 (Phase 2 Roadmap).

All sensor code is gated on `config.sensors.*.enabled: false` by default — no sensor functionality runs in Phase 1.
